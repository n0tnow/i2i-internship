Developer Features

This application also provides a REST API for developers.

API Endpoints:
  • GET /api/books - List all books
  • POST /api/books - Add new book
  • PUT /api/books/{id} - Update book
  • DELETE /api/books/{id} - Delete book
Tech Stack:
  • Spring Boot 3.5.3
  • Spring Data JPA
  • H2 Database
  • Thymeleaf + Bootstrap
